 # 📊 Super Store Sales & Forecasting Dashboard

---

## 🚀 Look Project Overview

This project showcases an interactive **Sales & Profit Analysis Dashboard** built using **Power BI**.

The dashboard provides deep insights into:

- Sales performance  
- Profit trends  
- Regional distribution  
- Shipping efficiency  
- 15-Day sales forecasting using time-series analysis  

The main objective of this project is to transform raw transactional data into meaningful business insights to support data-driven decision making.

---

## 📈 Key Business Metrics

- 💰 **Total Sales:** 1.6M  
- 📊 **Total Profit:** 0.2M  
- 📦 **Total Quantity Sold:** 22K  
- 🚚 **Average Shipping Days:** 4

## 📊 Dashboard Features

### 🔹 Sales Analysis
- Sales by Segment (Consumer, Corporate, Home Office)

- Sales by Sub-Category

### 🔹 Profit Analysis

- Year-wise Comparison
- State-wise Performance

### 🔹 Operational Insights
- Shipping Mode Analysis
- Payment Mode Distribution
- Average Delivery Time

### 🔹 Forecasting
- 15-Day Sales Forecast
- Confidence Interval Visualization

## 📌 Business Problem Statement

Retail businesses generate large volumes of transactional data.

The goal of this project was to:

- Identify high-performing regions and states  
- Analyze profitable product categories  
- Understand shipping performance  
- Detect monthly sales trends  
- Forecast short-term sales to improve inventory planning  

---

## 📊 Key Insights Derived

- 📍 California generates the highest sales revenue  
- 🚚 Standard shipping mode contributes maximum sales  
- 📱 Phones and Chairs are top-performing sub-categories  
- 📅 Sales show seasonal fluctuations across months  
- 📈 Forecast indicates steady growth trend for upcoming days  

---

## 💼 Business Impact

This dashboard can help management to:

- 📦 Optimize inventory planning  
- 🚛 Improve logistics performance  
- 💰 Focus on high-profit categories  
- 📊 Make data-driven strategic decisions  
- 🔮 Anticipate future sales using forecasting
